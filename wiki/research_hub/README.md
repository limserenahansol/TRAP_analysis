# Research LLM Wiki (cross-repo hub)

Markdown knowledge layer for **opioid behavior / TRAP / genelist** work. Source code stays in each code repository; **this folder is the canonical narrative on GitHub** (no separate `research-wiki` repo required).

- **Start here:** [index.md](index.md)
- **Changelog:** [log.md](log.md)
- **Rules (ingest / links):** [SCHEMA.md](SCHEMA.md)

Pattern inspired by Andrej Karpathy’s [LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f): raw facts live in code and data; the wiki is maintained as a separate, link-heavy layer.

## Optional standalone clone

You may keep a local-only copy under `behavior_task/research-wiki/` for Cursor; **public links** should target **this** `wiki/research_hub/` tree so they never 404.
